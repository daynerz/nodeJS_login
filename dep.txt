express: application server
ejs: templating language for diff views for login, register, etc.
--save-dev: development only
nodemon: restart server automatically every time changes are made
dotenv: environment variables to be stored in .env file
bcrypt: allow passwords to be hashed
passport (passport-local): local vers allows for email and password login
express-session: store and persist users in a session
express-flash: display error messages
method-override: override post method with delete for logout function
